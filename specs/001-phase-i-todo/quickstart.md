# Quickstart: Phase I Todo

## Prerequisites
- Python 3.13+
- uv (installed and in PATH)

## Running the App
1. Open your terminal in the `todo_app` directory.
2. Run: `python todo_app/main.py`

## Usage Examples
- Select `1` to add your first task.
- Select `2` to see your tasks.
- Data is NOT saved; if you exit, the list resets.
